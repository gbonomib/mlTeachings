K Nearest Neighbors¶

Source: James et al., An Introduction to Statistical Learning

This is what KNN at its simplest looks like:

$ \Pr\,(Y = j\, |\, X = x_0) = \frac{1}{K} \sum_{p\, \in \, N_K(x_0)}{\mathbb{1}(y_p = j)} $

Where:

  • $\mathbb{1}$ is the indicator function which takes value $1$ if the argument condition is true (0 otherwise).
  • $N_K(x_0)$ is the set of the $K$ dataset points nearest to $x_0$.

This formula means that the probability of an observation $x_0$ to belong to class $j$, is the weighed sum of the "votes" given by the dataset points of class $j$ among the $K$ closest to $x_0$.

That said, the more observations in $N_K(x_0)$ belong to class $j$ the more likely it is for $x_0$ to belong to class $j$ as well.

plot_model <- function(df, data_name, model){
    ## prepare prediction
    from <- 0
    to <- 1
    n <- 100
    grid_df <- expand.grid(
      x = seq(from, to, length.out = n),
      y = seq(from, to, length.out = n)
    )

    grid_df$prob = model %>% 
        predict(grid_df %>% select(x,y), type = 'prob') %>% select(prob = class_1)

    ## Prepare plot dataset

    data_df <- lapply(
      data_name,
      function(data_name) {
        df <- df[[data_name]] 
        rbind(
          df$full_train %>% mutate(partition = "train"),
          df$full_val %>% mutate(partition = "val")
        )
      }
    ) %>% bind_rows

    ## plot data
    ggplot() + 
      geom_raster(data = grid_df, aes(x = x, y = y, fill = prob), interpolate = T, alpha = 0.7) +
      scale_fill_gradient2(low = "blue", mid = "white",
                           high = "red", midpoint = 0.5, space = "rgb",
                           na.value = "grey50", guide = "colourbar") +
      geom_point(
          data = data_df, 
          aes(x = x, y = y, fill = 1- (as.numeric(factor(class)) - 1), alpha = partition), 
          shape = 21) +
      scale_alpha_manual(values = c("train" = 0.6, "test" = 1)) +
      theme_void() + ggtitle(model$method %>% toupper) +
      theme(legend.position = "none")
}

Kernels + a few things about distances¶

Ok we now understand the meaning of $\texttt{ks}$ (or, similarly, of $\texttt{kmax}$) ...but there are other two parameters ($\texttt{distance}$ and $\texttt{kernel}$) among the kknn ones...what are they?

In [32]:
models$kknn$parameters
parameterclasslabel
kmax numeric Max. #Neighbors
distance numeric Distance
kernel character Kernel

In the standard KNN algorithm all the neighbors have the same voting power in defining the output class. However in some cases may be very efficient to introduce a weighting scheme to give more discriminant power to the neighbors that are more similar to the target observation $x_0$ (i.e. those which are located at a closer distance d).

To parametrize the weighting scheme we introduce a so-called kernel function $g: \mathbb{R^+} \rightarrow \mathbb{R^+}$ that satisfies two conditions:

  • $g(d)$ gets its maximum for $d = 0$
  • $g(d)$ decreases monotonously as $d \rightarrow +\infty$

Using this kernel function we can predict the $class$ of $x_0$ as:

$ class = \operatorname{argmax}_j\left\{\sum_{K \in N_K(x_0)}{g(d(x_0, x_p))\mathbb{1}(y_p = j)} \right\} $

Or define the probability:

$\Pr(Y = j | X = x_0) = \frac{\sum_{p \in N_K(x_0)}g(d(x_0, x_p))\mathbb{1}(y_p = j)}{\sum_{p \in N_K(x_0)}g(d(x_0, x_p))}$

Where $d(x_1,x_2)$ is a distance function like the euclidean distance.

However, in different settings, an alternative distance function may turn out to be more effective.

The Minkowski distance is a generalization of the euclidean distance, parametrized by $P$:

$d\left(x,y\right)=\left(\sum_{i=1}^n |\,x_i-y_i\,|\, ^P\right)^{1/P}$

When $P = 2$ we have the euclidean distance, when $P = 1$ we have the manhattan distance and so on...

The choice of the distance function is a hyperparameter of the Kernelised K-Nearest Neighbor.

The other hyperparameter added by the KNN kernelisation regards the choice of kernel function $g$ itself.

These are three kernel function examples.

  1. Rectangular kernel (standard KNN kernel)
  2. Gaussian kernel
  3. Epanechnikov (alias Parabolic) kernel
